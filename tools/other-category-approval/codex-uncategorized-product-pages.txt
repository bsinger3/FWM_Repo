   categorize; no need to re-scrape the title.)
3. Suggest a mother category. PREFER one of the EXISTING categories. Only invent a NEW category
   if none genuinely fit, and keep it short + lowercase (e.g. "loungewear"). 'other' is NOT a
   valid answer — every row must get a real category or be flagged not_clothing=true. The whole
   point of this task is to get these rows OUT of the 'other' bucket.

EXISTING mother categories (prefer these):
   activewear, bodysuits, bottoms, dresses, intimates, jumpsuits, outerwear, sets, swimwear, tops, accessories
   Mapping hints: overalls/shortalls/rompers/playsuits/catsuits/boilersuits/flight-suits -> jumpsuits ;
   skirts/pants/shorts/jeans/joggers/leggings -> bottoms ; shirts/blouses/tanks/camis -> tops ;
   coats/jackets -> outerwear ; swimsuits/bikinis -> swimwear.
   NOTE: this site does NOT sell footwear. If a page is footwear (shoes/sandals/flip-flops) or
   non-apparel (yoga mat, gift card, shipping/protection add-on), set not_clothing=true instead of
   forcing a clothing category — those rows will be deleted, not categorized. Do NOT use 'other'.
